# [The Gifting Co.] 온라인 사전 과제

## Stack
- back-end: nest.js
- front-end: html / css / vanilla js


## 개발환경에 대한 설명
- OS: windows 11
- node version: 16.13.2
- npm version: 8.1.2

아래의 명령어로 node와 npm 버전을 확인해볼 수 있습니다.
```shell
node --version
npm --version
```

## 프로젝트 설치
yarn으로 구성된 프로젝트여서 yarn을 추천합니다.
### npm 사용자
```shell
npm install
```
### yarn 사용자
```shell
yarn install
```

## 프로젝트 실행
yarn으로 구성된 프로젝트여서 yarn을 추천합니다.  
`프로젝트 설치`를 먼저 진행해주세요.
### npm 사용자
```shell
npm start
```
### yarn 사용자
```shell
yarn start
```

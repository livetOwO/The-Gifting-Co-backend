# [The Gifting Co.] 온라인 사전 과제

## Stack
- back-end: nest.js
- front-end: html / css / vanilla js


## Getting started
```shell
# yarn 으로 의존성 패키지 설치
yarn install
# 빠른 실행
yarn start
# 프로덕션 환경 실행
yarn build
yarn start:prod
```
```shell
# npm 으로 의존성 패키지 설치
npm install
# 빠른 실행
npm start
# 프로덕션 환경 실행
npm run build
npm run start:prod
```
